zipapp进行打包

同一个目录建立文件hello.py 和 app.py,其中app.py引入hello.py的函数

返回上一级目录 python -m zipapp 包含app的目录名 -o first.pyz -m “app:main” ，然后就看到first.pyz

生产了 然后用python first.pyz 运行
